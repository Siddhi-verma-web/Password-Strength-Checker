## ✨ Features

- ✅ Real-time password strength analysis
- ✅ 7-bar animated strength indicator
- ✅ Color-coded feedback (Weak → Very Strong)
- ✅ 5 security rule checks with live checkmarks
- ✅ Show/Hide password toggle
- ✅ Animated canvas background (stars, aurora waves, 3D grid)
- ✅ Floating shield with orbit animation
- ✅ Fully responsive design

## 🛡️ Security Rules Checked

| Rule | Description |
|------|-------------|
| 🔢 Length | Minimum 8 characters |
| 🔠 Uppercase | At least one A-Z |
| 🔡 Lowercase | At least one a-z |
| 🔣 Number | At least one 0-9 |
| ⚡ Special | At least one !@#$% etc |

## 🎨 Tech Used

- **HTML5** — Structure
- **CSS3** — Glassmorphism card, neon glow, animations
- **JavaScript** — Regex-based strength logic, Canvas animation
- **Google Fonts** — Inter font
- **HTML Canvas API** — Animated background (stars, waves, grid)
